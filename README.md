# kakufile

My dotfile settings.

## Prerequisite

- git
- curl
- zsh
- tmux >= 2.4 (For oh-my-tmux)

### For vim/nvim
- vim >= 8.1, nvim >= 0.5
- node (For coc.nvim)
- clangd (For coc-clangd)
- universal-ctags
- cscope

## Install
```shell
git clone https://github.com/Dephilia/kakufile.git ~/.kakufile
cd .kakufile
./install.sh
```
