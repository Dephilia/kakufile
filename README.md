# kakufile

My dotfile settings.

## Prerequisite

- git
- curl
- zsh

## Install
```shell
git clone git@github.com:Dephilia/kakufile.git ~/.kakufile
cd .kakufile
./install.sh
```

## Configure list

### Program

- tmux (Using [oh-my-tmux](https://github.com/gpakosz/.tmux))
- vim (Using [vim-plug](https://github.com/junegunn/vim-plug))
- nano
- git
- alacritty
- Brewfile
