# kakufile

My dotfile settings.

## Prerequisite

- git
- curl
- zsh
- tmux >= 2.4 (For oh-my-tmux)
- vim >= 8.2
- gtags, ctags

## Install
```shell
git clone https://github.com/Dephilia/kakufile.git ~/.kakufile
cd .kakufile
./install.sh
```

## Configure list

### Program

- tmux (Using [oh-my-tmux](https://github.com/gpakosz/.tmux))
- vim (Using [vim-plug](https://github.com/junegunn/vim-plug))
- nano
- git
- alacritty
- Brewfile

- functions
